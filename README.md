# Simple Telegram Bot with Python

## Setup

1. Go to your Project.

    cd telebot

2. Create Virtual Environment.

    python3 -m venv env

3. Go to your Virtual Environment.

    source env/bin/active

4. Install requests.

    pip install requests

5. Set Bot Token at main.py

    token = '< BOT_TOKEN >'

6. Run program

    - start bot at Telegram
    - python main.py

### Have fun :))
